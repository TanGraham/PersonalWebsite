About Me

I was born and raised in Langley, BC, in a house filled with artifacts of the late 90s tech boom while being surrounded by expansive farm land. This setting cultivated an urge to build and fix and an interest in all things electronic. The decision to enter UBC computer engineering was only natural. Since then, this urge has been satisfied with projects such as an ecrypted web chat application, a hand build wooden surfboard (that actually floats), and my final year of university project: a drone detection system. Besides school, I enjoy snowboarding, surfing, biking and camping. The latter two culminating in a recent self supported bike trip to California. Scroll further to learn more.

Drone Tracker

A group project developing software for detecting and logging the presence of drones in an area using a software defined radio. The intended use of the system is combating the use of drones in restricted areas. The development process followed the scrum methodology and included extensive documentation.

The project uses Matlab, Simulink, Python and Latex

Burn Chat

A highly secure web application for multi party chat, this project was a practice in focussing on the security requirements of a system. The application uses client to client encryption and multi party key sharing while not requiring extra effort from the user.

The application is written in javascript, HTML, and CSS and uses nodeJS.
